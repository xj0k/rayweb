## 为什么需要一套标准流程

每次做竞品分析，我都面临同样的问题：信息太多、时间太少、分析框架不一致。

引入 AI 之后，我花了几个月打磨出一套稳定的流程。今天分享给大家。

## 完整流程

### Step 1：信息收集

```
你是一名产品分析专家。我需要分析竞品 [产品名]。
请从以下维度帮我梳理公开信息：
1. 核心功能与定位
2. 目标用户群
3. 定价策略
4. 近期重要更新（最近6个月）
5. 用户评价中的主要痛点
```

### Step 2：对比分析

收集完各家信息后，让 AI 做结构化对比：

```
以下是我收集的竞品信息：[粘贴信息]

请用表格形式对比以下维度：
- 核心价值主张
- 差异化特性
- 定价区间
- 主要缺点

然后给出：哪个竞品对我们威胁最大，理由是什么？
```

### Step 3：机会识别

```
基于以上竞品分析，我们产品的定位是 [简述]。
请帮我识别：
1. 竞品普遍存在的空白点
2. 我们可能的差异化机会
3. 需要警惕的竞争风险
```

## 注意事项

- **不要盲目信任 AI 给出的数据**，数据必须自己核实
- AI 擅长结构化整理，但对行业 context 的理解有限
- 每次用完记得保存 Prompt，持续迭代优化

## 总结

这套流程帮我把竞品分析时间从 2 天压缩到 4 小时。关键不是让 AI 替你思考，而是让它帮你快速处理信息，把精力留给判断。
