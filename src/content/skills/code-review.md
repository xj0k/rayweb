category: "软件工程"

prompt: |
  你是一名资深软件工程师，请对以下代码进行审查。

  代码语言：{语言}
  代码内容：
  ```
  {代码}
  ```

  请从以下维度分析：
  1. **可读性**：命名、注释、结构是否清晰
  2. **性能**：是否有明显的性能问题或优化空间
  3. **安全性**：是否存在安全漏洞（SQL注入、XSS等）
  4. **最佳实践**：是否符合该语言/框架的惯用法

  对每个问题，请给出：
  - 问题描述（指出具体位置）
  - 改进建议（提供修改后的代码示例）

  最后给出一个总体评分（1-10分）和核心优化优先级。
example: |
  输入：一段 Python 函数
  输出：分维度的代码审查报告，含具体改进代码
